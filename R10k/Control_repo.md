# Control Repository #

The control repository holds environment metadata, arranged for a single Puppet environment:

      example-control/
      ├── dist
      │   ├── profiles
      │   └── roles
      ├── environment.conf
      ├── hieradata
      │   └── ...
      ├── manifests
      │   └── site.pp
      └── Puppetfile
